# STM32 Autonomous Robot

Master thesis project.

Mobile robot platform based on STM32 microcontroller with LiDAR and sensor fusion for opponent detection and navigation.

## Hardware

STM32F411 Blackpill  
RPLIDAR A1M8  
HC-SR04 ultrasonic sensors  
Sharp IR sensors  
L9110 motor drivers  

## Features

- autonomous navigation
- opponent detection
- LiDAR scanning
